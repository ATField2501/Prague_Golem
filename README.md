PRESENTATION:

Bot IRC ecrit en python utilisant les librairies irclib et ircbot de Joel Rosdahl <joel@rosdahl.net>.


UTILISATION:

Lancer le bot à l'aide du script Golem_start.sh
La connection se fait par default sur les serveurs d'Epiknet. Le bot ouvre un canal #test et un canal #lymbes. Ce dernier est utiliser pour prendre des ordres du maitre et faire un conte rendu d'activitée, seul le maitre y a acces.


ATTENTION:

Lorsque le bot à pris vie, vous devez le tuer manuelement sur IRC avec la commande <!die> avant de fermer la boucle du script Golem_start sous peine de creer des processus fantome.
